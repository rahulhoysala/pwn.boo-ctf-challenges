# pwn.boo-ctf-challenges
Challenges created by me for the [pwn.boo](https://pwn.boo) CTF hosted by [Cyber Info](https://cyber.info)


Some of the challenges here cannot be solved indepedently because the infrastructure used to support it at the time of the CTF is no longer live. Therefore, much of the writeups will be more descriptive of the solution. 

These challenges were largely intended for beginners, with a few intermediate and advanced challenges to keep more experienced players engaged.
